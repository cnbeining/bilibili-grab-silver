# bilibili-grab-silver
Auto grab Bilibili's live silver

Usage
------

    python autograb.py (-h) (-c) (-u) (-l)
    
    -h: help:
    This.
    
    -c: cookies:
    location of cookies
    
    -l: Default: INFO
    INFO/DEBUG

Requirement
-------

- Python 2.7 OR

- Python 3.x, I don't really care.

- requests
