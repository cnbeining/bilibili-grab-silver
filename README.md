# bilibili-grab-sliver
Auto grab Bilibili's live sliver


Usage
------

    python autograb.py (-h) (-c) (-u)
    
    -h: help:
    This.
    
    -c: cookies:
    Location of cookies
    
    -u: Uploader:
    t: Tietuku
    i: Imgur

Requirement
-------

- Python 2.7

- requests

- BaiduOcr (https://github.com/Linusp/baidu_ocr)
